bd - carpeta que contiene los snipets o querys de la bd
    listos para ejecutar

clases - contiene todas las clases de cada modulo, con sus 
    respectivos metodos de crud

css - contiene los scripts css que serviran para dar
    estilos personalizados en caso de ser util

img - contiene todas las imagenes que necesitemos agregar a nuestras vistas

archivos - contiene todas las imagenes de los productos agregados

js - contiene todos los scripts js para cada modulo para ejecutar
    las funciones crud 

librerias - contendra librerias para la creacion de pdf, jquery, alertify
    bootstrap y select 2

procesos - contiene archivos intermedios entre las clases y las vistas 
    estos mandaran informacion entre una y otra cosa.

vistas - contendra todas las vistas de cada uno de los modulos
